Engagement - Responsive Pages

1. Why do we need a viewport meta tag?

= A viewport meta tag is important because it is what gives the ability to adjust a webpage depending on the device being used to view it like a mobile phone or a desktop. This is also ensures that you wont have to zoom in or scroll horizontally.

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

= If the same tag is used twice in CSS, the one that was written last will take priority. However if the word "!important" is used then that would override it, regardless of the order.

3. How do you change the CSS for a web page displayed on a phone?

= A way that we were taught in class to change the CSS for a webpage displayed on a phone is by using CSS media queries. This can be written at the end of the CSS, for example you can write "@media only screen and (max-width: 600px){ nav{ background-color: #222; flex-direction: column; } }". This would change the css code ONLY if the width of screen exceeded 600 pixels.
