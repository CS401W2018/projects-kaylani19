1. What is JavaScript? How did it get its name?

= It was a marketing ploy to name the prgramming language "Javascript". Sun Microsystems played a huge part in popularizing it because the name was similar yet JavaScript was unrelated to their programming language, Java.

2. What are the two ways you can run JavaScript?

= Two ways to run JavaScript:

  1. Browser:
    • It can be run in the browser, where it interacts with DOM
    • DOM represents HTML elements for example, D stand for Documents (e.g. the HTML file) and O represents objects (e.g. tags or elements in HTML)
  2. Server:
    • Was said to have been developed within the last decade, where JavaScript can be run outside of the browser using "Node.js"
    • Node.js is a runtime environment that allows JavaScript to be used for server-side programming.

3. What was the most popular front-end js library?

= Back in the browser wars, the most popular front-end JQuery.js library. After Google chrome came, nobody really used JQuery.js

4. What is the most popular front-end js library today?

= The most popular front-end js library is done by Meta, which is the parent company of Instagram and Facebook, they use React.js.
