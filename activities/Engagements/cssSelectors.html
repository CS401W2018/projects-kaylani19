= ID, class & div


What is the difference between an ID and a Class?

= ID uses only one HTML element and can only be used once; however, a class is used to target multiple elements and can be reused.


What is the difference between a SPAN tag and a DIV tag?

= A SPAN tag is an inline element that is used for small pieces of content. Whereas a DIV tag is a block element that's used for larger sections of content and starts on a new line.


What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

= A DIV tag is a generic container that doesn't have any purpose until given specific purpose through css styling. However, header, main and footer are elements that are used to help structure a webpage and more likely used than div
