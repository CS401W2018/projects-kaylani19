Why is the viewport meta tag important?

= A viewport meta tag is important because it is what gives the ability to adjust a webpage depending on the device being used to view it like a mobile phone or a desktop. This is also ensures that you wont have to zoom in or scroll horizontally.

How can you adjust the CSS for printing?

= To adjust the CSS for printing you can use styles that are good for printing like using properties that make it easy to read and to look organized when printed; this can include changing font sizes and colors.

When would you want to adjust the CSS for printing?

= You would want to adjust the CSS for printing when the webpage is the type that users would typically download and print such as articles or forms.
