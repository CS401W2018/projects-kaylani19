What is AJAX?

= AJAX also known as Asynchronous JavaScript and XML and lets web pages update parts of the content without reloading the whole page. It uses JavaScript to send and receive data from the server in the background, often in JSON format.

Why do you put your JS Code in a separate file?

= Moving JavaScript into a separate file, like form.js, keeps the code organized and easier to manage. It makes your HTML cleaner, allows reusability across pages, and enables browsers to cache the JavaScript file, which speeds up website performance.
