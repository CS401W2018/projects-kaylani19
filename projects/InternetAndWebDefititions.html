1. Internet

The internet is a worldwide network of computers connecting to one another; this ensures easier access for people to find, share and receive information.

2. Web

The web is all the documents, resources, information and data interconnected through the internet using web browsers.

3. How are the internet and web different?

The internet is a global system of connected computers, whereas the web is the service of accessing a collection of information through the internet.

4. ASCII and Unicode

ASCII - is the acronym for American Standard Code for Information Interchange. ASCII is a character encoding that uses 7-bit encoding which limits to represents only 128 characters.

Unicode - is an international character encoding standard that can represent more than 145,000 characters in different languages

5. What is UTF-8

UTF is an encoding system for Unicode and is compatible with ASCII.

6. URL

URL is the acronym for Uniform Resource Locator and is basically the address displayed for a web page.

7. Domain Name

A domain name is/are words or phrases that are associated with a web page and makes it easier to find a specific web page without having to memorize its entire address

8. HTTP

HTTP is a protocol used to transfer information over networked devices. Hypertext Transfer Protocol requests and shows the web page when a user entered a URL.

9. HTTPS

HTTPS is just a more secure version os HTTP

10. IP Address

An IP address is a unique numeric identifier that is assigned to every device that is connected to a network.

11. DNS

DNS is an acronym for Domain Name System and is basically a system set up for domain names to make it easier for computers to find a specific web page. DNS is sometimes compared to a phone book.

12. W3C Recommendations

W3C recommendations are guidelines to promote consistency and ensure that the best web developments such as compatibility and accessibility for different devices.

13. XML, XHTML, HTML, HTML5

XML - is a file format system that helps define, store and transport data.

XHTML - is a more strict version of HTML that follows XML rules and is said to be more effortless for diverse sets of devices ranging from cellphones to desktops.

HTML - is the standard language used to create web pages

HTML5- is the latest version of HTML.
